🌐 **[Visitar el sitio web del taller aquí](https://cbarriosperez.github.io/Taller_IndicesAgroclimaticos_Frijol)**

## 📖 Acerca del Proyecto

El cultivo de frijol es fundamental para la seguridad alimentaria, y su rendimiento está estrechamente ligado a las condiciones climáticas. Este sitio web fue creado como una herramienta de apoyo y guía interactiva para facilitar la enseñanza y el aprendizaje sobre cómo el clima afecta a este cultivo.

A través de este material, los participantes del taller pueden acceder a conceptos teóricos, metodologías y ejercicios prácticos para calcular, analizar e interpretar distintos índices agroclimáticos (como requerimientos hídricos, grados día de desarrollo, estrés térmico, entre otros) aplicados específicamente a las fases fenológicas del frijol.

## 🎯 Objetivos del Taller
- **Comprender** la importancia de las variables climáticas en el desarrollo del cultivo de frijol.
- **Calcular e interpretar** índices agroclimáticos clave utilizando datos reales.
- **Aplicar** herramientas computacionales para el manejo y visualización de datos climáticos orientados a la agricultura.
- **Apoyar** la toma de decisiones agrícolas basadas en información climática.

## 📚 Contenido del Sitio Web
El sitio está estructurado de la siguiente manera:
1. **Introducción:** Conceptos básicos de agroclimatología y fenología del frijol.
2. **Datos Climáticos:** Fuentes de información, descarga y procesamiento previo.
3. **Cálculo de Índices:** Fórmulas y aplicación práctica de índices (térmicos, hídricos, etc.).
4. **Visualización y Análisis:** Gráficos y mapeo de los resultados para una mejor interpretación.
